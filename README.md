# Controle de Estoque - Frontend

Frontend desenvolvido em **Angular** para consumo da API de controle de estoque.

## Objetivo
Este projeto foi criado para fornecer a interface do sistema de controle de estoque, permitindo a interação com dados de:
- produtos
- clientes
- fornecedores
- vendas

## Funcionalidades
- Listagem de produtos
- Cadastro de produtos
- Edição e exclusão de registros
- Consulta de informações do sistema
- Integração com API backend
- Interface para operações de estoque

## Tecnologias utilizadas
- Angular
- TypeScript
- HTML
- CSS

## Como executar localmente

### Pré-requisitos
- Node.js
- Angular CLI
